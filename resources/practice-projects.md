# Practice Projects

## Beginner Projects

### 1. Calculator
Build a calculator with TypeScript classes and methods.

### 2. Todo List
Create a todo list application with TypeScript interfaces and classes.

### 3. Student Grade Tracker
Track student grades with arrays, objects, and type safety.

## Intermediate Projects

### 4. Library Management System
Use classes, inheritance, and interfaces to manage books and users.

### 5. API Client
Create a type-safe API client using generics and async/await.

### 6. Event System
Build an event emitter using TypeScript classes and generics.

## Advanced Projects

### 7. Type-Safe ORM
Create a simple ORM with generics, conditional types, and mapped types.

### 8. State Management Library
Build a state management system using TypeScript advanced types.

### 9. Form Builder
Create a dynamic form builder with type-safe validation.

## Full Stack Integration

After completing this course, integrate TypeScript with:
- **Frontend**: Next.js (fs-course-frontend)
- **Backend**: Express.js (fs-course-backend)
- **Database**: Prisma (fs-course-database)
